使用 iWeb、Pages 和可用于 Mac OS X 的许多其他应用程序来快速和轻松地创建和发布您自己的网站。

从 Mac 创建和发布您自己的网站是如此迅速！当您的站点准备好时，发布它是很轻松的事情。

打开“系统偏好设置”，点按“共享”，然后选择“Web 共享”。

您的操作就完成了。您的站点现在可以通过家里或工作地点的专用网络访问。

如果您的电脑已连接在 Internet 上，它还可以被各个地方的朋友访问。只需要向他们发送“共享”偏好设置中所显示的地址。

Apache 功能

“Web 共享”构建在 Apache Web 服务器之上，这是 Mac OS X 附带的一种行业标准技术。有关 Apache Web 服务器的更多信息，请参阅 Apache 手册。
